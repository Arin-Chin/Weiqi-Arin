# WeiQi · 围棋

一个轻量级的**围棋打谱与对弈工具**，纯前端实现，打开浏览器即可使用。

## 功能特性

- **对战模式** — 黑先白后轮流落子，支持完整的提子、打劫规则
- **打谱模式** — 自由放置黑白棋子，适合棋局复盘与研究
- **终局数子** — 提交终局 → 标记死子 → 自动计算胜负
- **贴目设置** — 支持中国规则（3¾子）、日韩规则（6.5目）、无贴目
- **悔棋与重置** — 随时撤销上一步或重新开始
- **统计面板** — 实时显示黑白双方在盘数与吃子数
- **悬停预览** — 落子前预览棋子位置，减少误点

## 使用方式

直接用浏览器打开 \`index.html\`，无需任何依赖或服务器。

\`\`\`
git clone https://github.com/ArinChin/WeiQi.git
cd WeiQi
start index.html
\`\`\`

## 技术栈

- 原生 JavaScript（ES6 Class）
- HTML5 Canvas 渲染
- 无任何第三方依赖
